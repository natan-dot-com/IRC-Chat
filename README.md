## How to run

Tested in Debian 11 '_Bullseye_' and Windows 11. Compiled with G++11.

```bash
# Compila tanto o servidor, quanto o cliente.
make

# Roda o sevidor
./build/server/main

# Roda o client
./build/client/main 8080
#                   ^^^^~~~ porta para se conectar.
```
